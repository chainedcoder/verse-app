# Verse

Verse is a beautiful web application for discovering, sharing, and downloading poetry and lyrics. 
Features include a responsive layout, beautiful typography, dark/light themes, and the ability to export poems into beautifully styled, customized layout cards (minimal, dark, love letter, and story formats).

## Development

To run the app locally, start a local HTTP server in the root directory:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080` in your browser.
